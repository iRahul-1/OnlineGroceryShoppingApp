Online Shopping Application README
Description
This project is an online shopping application implemented in Java. It allows users to select products such as potatoes, onions, sugar, and oil, add them to a cart, view the cart, and place orders. The application is designed to run in a console-based environment.

Features
Users can select from a range of available products.
Users can add products to their cart.
Users can view the items in their cart.
Users can place orders for the items in their cart.

Usage
To use the application, follow these steps:

Run the OnlineShoppingApp class.
Select the desired product by entering the corresponding digit.
Follow the prompts to add items to the cart, view the cart, and place orders.
Classes
OnlineShoppingApp: Contains the main method and orchestrates the online shopping process.
User: Represents a user of the application and provides methods to interact with the shopping cart.
Product: Represents a product available for purchase.
Dependencies
This project has no external dependencies.

Setup
To run the project locally, make sure you have Java installed on your system.
